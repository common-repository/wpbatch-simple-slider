== Installation ==



1. Upload `wpbatch-simple-slider.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will see a new menu in the dashboard "Simple Slider"... add some images and texts from there
4. now just add this shortcode anywhere in the page, post or widgets 
<pre>[simple_slider]</pre>
or Place <pre><?php do_shortcode('[simple_slider]'); ?></pre> in your templates php files

Good Luck
